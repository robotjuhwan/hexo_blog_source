# INTRODUCTION
처음부터 적혀있다. HTML and CSS 를 배우는 것은 어렵다. 그러나 그럴 필요가 없다. 감사합니다!

이 모든 과정이 다 입문과정이라고 한다. 그렇기에 더더욱 기본이 되는 개념일 듯

아래가 약자를 풀어쓴 것이다.

* HyperText Markup Language (HTML)
* Cascading Style Sheets (CSS)

또 여기에 기본 요소 중 하나인 

* JAVASCRIPT
가 포함된다. 이 세 요소는 아주 다르면서도 웹이 대부분 이 세가지로 형성 된다고 한다.

이 세가지의 차이점을 각각 알고있으라고 하는데 음. 그렇다.

대략 생각해보면 HTML은 웹의 내용을 구성하고, CSS는 스타일을 규정하고, JAVASCRIPT는 Function을 형성한다. 대략 이렇게 지금까지는 이해했다.

HTML에 대한 Basic을 좀 찾아보니 솔직히 아래 영상한번 보고 vscode로 따라해보니까 모든게 대략 이해가 됐다.
