# 잡담
리액트를 왜 하는가

항상 임베디드쪽을 하면서

node.js에 대한 약간의 동경이 생겼었다. 조금은 할 수 있어야하지 않을까?

흔히 개발자라고 하면 다양한 종류가 있겠지만, 그 중에서도 웹 개발자가 굉장히 많을 것이다.

약간 트렌디한 느낌. 맥북으로 코딩할 수 있는 느낌

조금 만들어보고 싶은 사이트도 있고하니, node.js나 react.js를 좀 체험해봐야겠다.

다섯가지 필요사항은 다음과 같다.

* ES6 syntax and features에 대한 친숙함 - > [학습 링크](https://www.taniarascia.com/es6-syntax-and-feature-overview/)
* Node.js and npm 가 설치된 환경 - > [학습 링크](https://www.taniarascia.com/how-to-install-and-use-node-js-and-npm-mac-and-windows/)

음. 무작정 웹개발을 한번 해보겠다고 생각했는데, 생각보다 가져야 할 기본 개념이 나의 과거와 너무 맞지 않았다.

약간 기본부터 맛보기를 많이 해봤어야 하는 느낌.

일단 HTML에 대해서 공부를 막 해본적은 없다. CSS도 잘 모르겠고.

순서대로 다 학습이 필요할 것 같다.

친절하게도 학습할 수 있는 링크가 다 제공되어 있다 ㅠㅠ 차근차근 한번 읽어보고 기본을 다져봐야겠다.
