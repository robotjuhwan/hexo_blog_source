# PouL Blog Working Notes

Before writing, revising, or publishing a post, read `BLOG_STYLE.md` and follow
its content, visual, and public-safety rules.

For portfolio or market posts, only use verified data and state the snapshot
time and scope. Never publish credentials, tokens, account numbers, exact
balances, share quantities, order identifiers, or absolute local paths.

The comment widget is disabled unless both `WALINE_SERVER_URL` and
`WALINE_TURNSTILE_SITE_KEY` are supplied at build time. Do not add placeholder
values to tracked configuration files.
